Apr 7th, 2018 7:24 pm

虽说 CocoaPods 有 cocoapods-packager 这个插件可以生成二进制版本，但是这个库的维护者似乎并不是很活跃，很多 issue 和 pr 过了挺久了还堆积在那里。于是我决定试试 Carthage ，不过不利用 Cartfile 生成依赖，还是用的 CocoaPods 那一套。

要让组件支持 Carthage ，只需要工程里有一个 shared framework target 即可。那么针对 CocoaPods 生成的工程，我们先满足 framework target 这个要求，这个只需要在 Podfile 里面设置 use_frameworks! 就可以了。
