如何愉快地使用命令行

Mar 19th, 2017 7:08 pm

古人云：

 工欲善其事，必先利其器

iTerm

命令行字符操作快捷键：
功能 快捷键
到行首 ctrl + a
到行尾 ctrl + e
前进后退 ctrl + f/b
会和tmux默认的prefix key冲突，基本不用
上一条命令 ctrl + p
搜索历史命令 ctrl + r
后面默认会和fzf结合
前向/后向删除字符 ctrl + d/h
前向/后向删除单词 ctrl + w/alt + d
清除当前行/到行尾 ctrl + u/k
交换光标处文本 ctrl + t


需要留意的是，上面的快捷键不仅仅适用于 iTerm ，其中的绝大多数快捷键也可以在 mac 自带终端以及 Linux 的终端上使用。

功能操作快捷键：
功能 快捷键
清屏 command + r
对tmux有干扰，基本不用
回放操作历史 command + option + b
剪切板历史 command + shift + h
查看历史命令 command + ;
搜索内容 command + f
输入内容后 + (tab / tab + shift = 前向 / 后向推移)
内容会默认拷贝至剪切板
