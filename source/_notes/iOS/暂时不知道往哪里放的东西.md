### NSCache 和 NSMutableDictionary

NSCache 常用来做缓存，相比 NSMutableDictionary 有以下特点

- 发生内存警告时，可通过配置的策略自动清除一些对象
- 访问是线程安全的 （鉴于其清除对象时，是异步清除，所以需要保证这一点）
- 不会拷贝 key 对象（copy 操作），只会引用之



### 深拷贝和浅拷贝

深拷贝：拷贝对象内容

浅拷贝：拷贝对象指针

| 可变/不可变 | 操作 | 深浅拷贝 |
| ------ | ------ | ------ |
| 不可变 | copy | 浅 |
| 不可变 | mutableCopy | 深 |
| 可变 | copy | 深 |
| 可变 | mutableCopy | 深 |
